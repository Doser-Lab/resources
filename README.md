# Resources for Statistical Ecology and Forest Science Lab Members

+ `github_process.md`: describes process for initating a repository in the SEFS Lab GitHub page for different research projects. 
+ `Graduate_Student_Advising_Statement_SEFS_Lab.pdf`: graduate student advising statement for the SEFS Lab consisting of lab policies, what I expect from you, what you expect from me, and other information relevant to your graduate program. 
+ `NCSU_HPC_Guide.pdf`: tutorial on how to use the NC State High Performance Computing system (Hazel). 
+ `software_on_hpc.html`: guidance on how to set up your NC State HPC account for loading and installing software (including R packages). This allows all lab members to use the same set of R packages and should be done prior to using R on the HPC.
